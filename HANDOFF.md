### 1. トップページ (`http://localhost:3100`)

  - 👔 就活に悩む
  - 🌧️ もう詰んだ…
- Framer Motionによるアニメーション

### 2. カテゴリー別一覧ページ (`/categories/[slug]`)
- カテゴリーヘッダー（絵文字・名前・説明）
- 先人の声カード一覧
  - 先人の名前・年齢
  - キャッチコピー
  - コメント数・アーカイブ数
- 戻るボタン

### 3. 先人の声詳細ページ (`/stories/[id]`) ⭐
**ページめくりUI - 最も重要な機能！**

6ページ構成：
1. **導入**: 先人の基本情報とキャッチコピー
2. **状況の確認**: 「あの時、こんな状況でした」
3. **感情の掘り下げ**: 「あの時の気持ち」
4. **決断と意味**: 「転機が訪れた」
5. **ひとこと**: 最も重要なメッセージ（先人の名前付き）
6. **アクション**: 勇気ブック保存・コメント投稿ボタン

機能：
- スムーズなページ遷移アニメーション
- ページインジケーター（進捗ドット）
- 下スワイプボタン（バウンスアニメーション）
- 完了後は前のページに戻る

## 💬 よくある質問と解決方法

### Q1: 開発サーバーが起動しない

```bash
# node_modulesを再インストール
rm -rf node_modules pnpm-lock.yaml
pnpm install
pnpm dev
```

### Q2: ビルドエラーが出る

```bash
# キャッシュをクリア
rm -rf .next
pnpm dev
```

### Q3: コンポーネントが表示されない

- ブラウザのキャッシュをクリア（Cmd+Shift+R）
- 開発サーバーを再起動

### Q4: スタイルが反映されない

- `globals.css`が正しくインポートされているか確認
- TailwindCSSのクラス名を確認
