# senjin 画面設計・ユーザーフロー設計書

## デザインコンセプト

### テーマ
**「心に染み渡る、温かいページめくり体験」**

### デザインの方向性
- **エモーショナル**: 温かみのあるフォント、柔らかい配色、余白を活かしたレイアウト
- **モバイルファースト**: スマホでの快適な閲覧を最優先
- **読みやすさ**: 1画面1メッセージ、ゆったりとしたペース感
- **没入感**: スクロールではなくページめくりのようなUI

### カラーパレット

```
Primary Colors:
- メインカラー: #4A5568 (落ち着いたグレー)
- アクセント: #F6AD55 (温かみのあるオレンジ)
- テキスト: #2D3748 (読みやすい濃いグレー)

Background:
- ベース: #FFFBF0 (温かみのあるオフホワイト)
- カード: #FFFFFF (純白)

Category Colors:
- 進路: #667EEA (落ち着いた青)
- 転職: #F6AD55 (オレンジ)
- 就活: #48BB78 (グリーン)
- 困難: #FC8181 (柔らかいレッド)

Accent:
- いいね: #FC8181 (ハート色)

```

### タイポグラフィ


## 画面詳細設計

### 1. トップページ (`/`)

**目的**: サービスの世界観を伝え、悩みカテゴリーへ誘導する

**レイアウト**:
```
┌─────────────────────────────┐
│                             │
│        [ロゴ]               │
│       senjin                │
│                             │
│    あなたの悩みを           │
│    選んでください           │
│                             │
│  ┌───────────────┐         │
│  │  🎓 進路に悩む  │         │
│  └───────────────┘         │
│                             │
│  ┌───────────────┐         │
│  │  💼 転職・退職  │         │
│  │     に悩む      │         │
│  └───────────────┘         │
│                             │
│  ┌───────────────┐         │
│  │  👔 就活に悩む  │         │
│  └───────────────┘         │
│                             │
│  ┌───────────────┐         │
│  │ 🌧️ もう詰んだ… │         │
│  └───────────────┘         │
│                             │
│    [右上: ログイン]          │
└─────────────────────────────┘
```

**アニメーション**:
- ページ読み込み時: ロゴがふわっとフェードイン
- カテゴリーカード: 下から順番にスライドイン（0.1秒ずつ遅延）
- ホバー時: カードが軽く浮き上がる（transform: translateY(-4px)）

**実装ポイント**:
```tsx
// Framer Motionでアニメーション
<motion.div
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ delay: index * 0.1 }}
>
  <CategoryCard />
</motion.div>
```

**実装ポイント**:
```tsx
// ページめくりアニメーション
const variants = {
  enter: { opacity: 0, y: 50 },
  center: { opacity: 1, y: 0 },
  exit: { opacity: 0, y: -50 }
}

<AnimatePresence mode="wait">
  <motion.div
    key={currentPage}
    variants={variants}
    initial="enter"
    animate="center"
    exit="exit"
    transition={{ duration: 0.5 }}
  >
    <StoryPage content={pages[currentPage]} />
  </motion.div>
</AnimatePresence>
```

## ユーザーフロー

### メインフロー: 先人の声を読む（未登録ユーザー）

```
[トップページ]
     ↓
  カテゴリー選択
     ↓
[先人の声一覧]
     ↓
  キャッチコピーをタップ
     ↓
[先人の声詳細]
  ページめくりで読む
     ↓
  「勇気ブックに保存」をタップ
     ↓
  ⚠️ ログインが必要です
     ↓
[ログイン/サインアップ]
     ↓
  ログイン完了
     ↓
[先人の声詳細に戻る]
     ↓
  保存完了！
```

### サブフロー: コメント投稿

```
[先人の声詳細]
     ↓
  「コメントを残す」をタップ
     ↓
  （未ログインの場合）
  ⚠️ ログインが必要です
     ↓
[ログイン/サインアップ]
     ↓
[コメント投稿モーダル]
     ↓
  コメント入力
     ↓
  「投稿する」タップ
     ↓
  投稿完了！
  （詳細ページに反映）
```

### 管理フロー: 先人の声を登録

```
[管理ダッシュボード]
     ↓
  「新しい声を追加」
     ↓
[先人の声編集画面]
     ↓
  GPTsの回答を貼り付け
     ↓
  カテゴリー・キャッチコピー入力
     ↓
  プレビュー確認
     ↓
  「公開する」
     ↓
  公開完了！
  （一般ユーザーに表示される）
```
